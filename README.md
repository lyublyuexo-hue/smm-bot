# 🤖 SMM Бот — Инструкция по запуску

## Структура проекта

```
smm_bot/
├── bot.py              # Точка входа
├── config.py           # ⚙️ НАСТРОЙКИ (токен, ID админа, реквизиты)
├── database.py         # База данных (SQLite)
├── requirements.txt    # Зависимости
├── handlers/
│   ├── start.py        # /start и главное меню
│   ├── views.py        # Ветка «Просмотры»
│   ├── balance.py      # Ветка «Пополнить баланс»
│   └── admin.py        # Админ-панель + заглушки для Подписчиков/Лайков
└── keyboards/
    └── __init__.py     # Все клавиатуры
```

## Установка

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Отредактировать config.py:
#    - BOT_TOKEN    → токен от @BotFather
#    - ADMIN_ID     → ваш Telegram ID (узнать у @userinfobot)
#    - PAYMENT_DETAILS → ваши реквизиты для оплаты

# 3. Запустить бота
python bot.py
```

## Как работает пополнение баланса

1. Пользователь выбирает метод оплаты → бот показывает реквизиты
2. Пользователь отправляет фото чека
3. Вам в чат приходит фото + кнопки **✅ Подтвердить** / **❌ Отклонить**
4. При подтверждении — начисляется `DEFAULT_REPLENISH` (по умолчанию 50 000 сум)

> ⚠️ **Важно:** сумму пополнения нужно настраивать вручную в `balance.py` (переменная `DEFAULT_REPLENISH`),
> либо добавить диалог с вводом суммы от админа.

## Добавление новых услуг (Подписчики, Лайки)

Файл `handlers/admin.py` содержит заглушку. По аналогии с `views.py` создайте:
- `handlers/subscribers.py`
- `handlers/likes.py`

И подключите их в `bot.py`.

## База данных

SQLite-файл `smm_bot.db` создаётся автоматически при первом запуске.

Таблицы:
- `users(user_id, balance)`
- `orders(order_id, user_id, service_type, amount, price, link, status)`
